Multi-Coin Hardware Wallet

A secure, air-gapped hardware wallet for blockchain assets, designed and developed from scratch.
What is it?
This project is a custom-built hardware wallet supporting multiple cryptocurrencies. It is designed for maximum security, using air-gapped QR-based signing and hardware cryptography to protect private keys and transactions.
What I Did
  • Designed and developed the wallet using Raspberry Pi Zero as the main controller
  • Integrated a camera for QR code scanning and transaction signing
  • Implemented secure signing with the ATECC608A cryptographic chip
  • Built a Python desktop UI for user interaction
  • Focused on hardware cryptography, anti-tamper techniques, and secure key storage
Key Technologies Used
Raspberry Pi Zero
Camera Module
ATECC608A Crypto Co-Processor
Python (Desktop UI)
QR Code Signing
Hardware Security
